المشروع الأول: صفحة HTML تجريبية

هذه صفحة بسيطة لمشروع HTML توضح كيفية ربط الملفات واستخدام الأنماط (CSS).

محتويات المشروع

  • عنوان (h1) باللغة العربية
  • قائمة ارتباطات
  • نصوص تجريبية

مثال كود

<!DOCTYPE html>
<html lang="ar" dir="rtl">
  <head>…</head>
  <body>…</body>
</html>
